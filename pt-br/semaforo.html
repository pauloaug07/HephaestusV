Placa Arduino

O Arduino é uma placa de prototipagem eletrônica de código aberto (open-source) e hardware livre, agora, vamos fragmentar essa mensagem para realmente absorver qual é sua função. O Arduino, como dito anteriormente, é uma plataforma que possibilita o desenvolvimento de projetos eletrônicos, perfeito para a produção com baixo custo e tempo de desenvolvimento, sendo muito útil para desenvolvedores iniciantes. O Arduino possui tanto o Hardware quanto um Software, permitindo uma maior variedade na produção de projetos. Possui Hardware livre e Código Open-Source, permitindo que os desenvolvedores possam copiar, redistribuir com ou sem alterações, tanto o hardware quanto o software, que está disponibilizado de forma gratuita.

Projeto Tinkercad

Para o nosso projeto, estamos utilizando o software Tinkercad, segue resumo sobre o mesmo. O Tinkercad é um aplicativo online e gratuito desenvolvido pela Autodesk. Foi feito para a criação e design de modelos 3D, mas também é possível desenvolver circuitos eletrônicos na plataforma. Por ser intuitiva e fácil de usar, é um recurso excelente para alunos, professores e projetistas iniciantes. O Tinkercad possui bibliotecas para Arduino e Micro:bit integradas, facilitando ainda mais o desenvolvimento de projetos.
